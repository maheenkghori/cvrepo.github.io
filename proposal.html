Project Proposal

Hospital Management System

Description

We are providing a platform which will help patients to get to information of respective doctors, easily pay their outstanding amount and it will also help visitors to know about patients.Our aim is to build a web site which helps the hospital management to easily manage the doctor and patients in the hospital. It will help Patients , staff , doctors and visitors. It will make work easy for everyone. Our aim is to get this installed in at-least small clinics which may bring positive change.

Features

  • The system automates the manual procedure of managing hospital activities.
  • Doctors can view their patientâ€™s.
  • With the help of this system admin can easily see bed availability and immediately allot it whenever any patient arrives without manually looking for bed availability.
  • It even generates an instant bill.
  • The system is convenient and flexible to be used
  • It saves their time, efforts, money and resources
  • Doctors can register but cant login without admin approval.
  • Admin can manage the doctor and patient.

Core Domain Features

  • Outstanding bill printing
  • Appointment reservation with doctor
  • View Doctor details
  • Visitors can check information about patients
  • We use Skype calling software to maintain better communication between patient and doctors.
